# DC_101_Matching-Type_Paje-Ian-Joshua
Matchy Matchy is a fun and colorful matching game where players connect matching colors before time runs out! Each round adds more colors, making the challenge more exciting as you level up. With cute bubble-style visuals, playful sounds, and fast-paced gameplay, Matchy Matchy tests your memory, speed, and focus. How far can you go before the timer says Game Over?

Technologies Used
HTML
CSS
JavaScript

How to Run the Game
1. Download or clone the repository
2. Open index.html in any web browser
